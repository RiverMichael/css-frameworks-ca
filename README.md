## About The Project

[![Xplor screenshot][product-screenshot]][live-site]

### Description

This is my project for the course in CSS Frameworks at [Noroff](https://www.noroff.no). The assignment was to create a responsive social media application using Bootstrap and and customising styles with Sass.

## Getting Started

To get a local copy up and running follow these steps.

<!-- INSTALLATION -->

### Installation

1. Open the terminal and go to your desired folder

```sh
cd replace-with-your-desired-folder
```

2. Initialize a local Git repository

```sh
git init
```

2. Clone the repo:

```sh
git clone https://github.com/RiverMichael/css-frameworks-ca.git
```

3. Navigate to the project folder

```sh
cd name-of-the-project-folder
```

4. Install NPM packages

```sh
npm i
```

5. Start watch and live-server to view the project live

```sh
npm run watch
```
